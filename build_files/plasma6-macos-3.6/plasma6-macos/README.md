Transform your KDE Plasma 6 desktop into a refined macOS-inspired environment using a single, automated script. Designed for reliability and consistency across major KDE distributions, this tool handles the full customization process end to end—no manual tweaking required.

## Supported Systems

- Fedora KDE Plasma  42/43    
- KDE Neon    
- Manjaro Linux KDE Plasma    
- openSUSE Tumbleweed    
- MX Linux 25 KDE    
- Arch Linux with KDE Desktop   

## Features

### Complete macOS-Style Integration

Automatically installs and configures Plasma themes, icon sets, fonts, widgets, wallpapers, cursor themes, and system settings to deliver a cohesive macOS-like appearance.

### Polished User Experience

Optimized layouts, consistent visual language, and smooth animations provide a clean, modern desktop that remains performant and practical for daily use.

## Included Files

### Installation Scripts

- `install.sh` — fully automates KDE Plasma 6+ customization    
- `tweaks.sh` — switches Light/Dark modes for Plasma and GTK/Libadwaita themes    

## Usage

1. Extract the archive:    
    ```bash
    unzip plasma6-macos-3.6.zip
    ```
    
2. Enter the directory:    
    ```bash
    cd plasma6-macos/
    ```
    
3. Run the installer:    
    ```bash
    ./install.sh
    ```
    

The system is transformed immediately after execution. The result is a clean, macOS-inspired KDE Plasma 6 desktop with minimal effort and full reversibility.
